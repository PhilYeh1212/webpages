AI 相關專案與成果

核心AI專案：無人機影像智慧辨識系統 (艾思特能源)

主導開發用於無人機影像的智慧辨識系統。負責內容包括機器學習模型選型、資料集處理、模型建構、訓練與驗證，以及系統部署。 此系統成功實現對特定工業設施或環境目標的自動化偵測與分類，有效提升檢測作業的效率與準確性。

技術棧：機器學習 (Machine Learning), 深度學習 (Deep Learning), Python, TensorFlow/PyTorch, 電腦視覺 (Computer Vision), 影像處理

自動化資料處理與分析系統 (支援AI應用)

運用 Python 與 MySQL 開發及維護自動化數據蒐集、清洗與處理流程，為後續的數據分析及 AI 模型訓練提供高質量的資料基礎。 優化資料庫效能，確保數據處理的效率。

技術棧：Python, MySQL, 資料 ETL, 自動化腳本, 數據分析前期處理

工業物聯網(IIoT)與AIoT數據整合方案

整合 RS485, Modbus RTU/TCP, Ethernet/IP, HTTP Request, TCP/IP Socket 等工業通訊協議， 建立穩定的數據採集通道，為將AI演算法部署於邊緣端或雲端提供了基礎數據流與控制介面。

技術棧：Modbus, Ethernet/IP, RS485, TCP/IP, Python, 物聯網(IoT)數據整合

支援AI服務的後端API系統開發

設計並實作高效能的後端API，用於AI模型輸入資料的接收、預處理，以及模型預測結果的輸出與展示。確保系統的穩定性與可擴展性，支持前端應用與AI核心功能的資料互通。

技術棧：Python (Flask/Django), RESTful API, 資料庫整合, Web開發
